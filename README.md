# StorePageToDB
This program fetches all products from a specific CBD seller and puts them into a database. This was a test for utilizing Celery Workers to make fetching many pages and scraping significantly faster.

The owner of a family owned retail store wished to change from squarespace to Bluebutton. However, there was no way to easily pull out their data. This program automates this process and used celery workers to make if not run on their local network clogging up their traffic.
